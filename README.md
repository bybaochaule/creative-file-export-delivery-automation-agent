# Creative File Export & Delivery Automation Agent

## Overview

This skill helps an agent design, audit, and improve professional creative file delivery systems for digital products, client handoffs, marketplace uploads, and creative asset packages.

It focuses on folder structure, file naming, export settings, preview/thumbnail rules, README and license placement, ZIP packaging, automation-safe SOPs, and quality-control workflows.

## When to use

Use this skill for Etsy, Creative Market, Gumroad, PromptBase, Creative Fabrica, Canva/template delivery, SVG/PNG/PDF/PSD/AI/MP4 export systems, photography/video client delivery, branding-kit handoffs, digital product audits, and final ZIP packaging workflows.

## Not for

Do not use it for malware, hidden scripts, deceptive packaging, fake licenses, rights clearance, legal advice, marketplace-rule bypassing, or packaging assets the user is not authorized to distribute.

## Package contents

- `SKILL.md`: Main agent instructions.
- `README.md`: Human-readable package overview.
- `agents/openai.yaml`: Display metadata and invocation policy.
- `references/style-guide.md`: Extended style notes and example expectations.
- `assets/.gitkeep`: Placeholder for reusable templates/assets.
- `scripts/example_helper.py`: Safe placeholder helper.
